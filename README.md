## 주요 단계
1. 데이터 차원
    1. 정규화(스케일링이 주는 영향력 제거)
    2. 특성 선택(corr을 보거나 SFS로 기계적 특성 선택)
2. 알고리즘(모델 차원)
    1. AutoML (Pycaret)
    2. 좋은 결과를 낸 모델을 baseline 삼아서 Hyperparameter Tuning(Optuna) 
3. Optuna Dashboard
